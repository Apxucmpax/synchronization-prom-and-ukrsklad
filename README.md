### How do I get set up? ###

* NodeJS v13.13.0
* Для коректной работы нужно вставить fbclient.dll в папку
* '\node_modules\firebird\build\Release' для 32 битной ноды
* 32 битный драйвер, для 64 битной 64 битный драйвер
